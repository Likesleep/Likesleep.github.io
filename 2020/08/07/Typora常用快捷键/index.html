    Typora常用快捷键
    发表于 2020-08-07| 更新于 2020-08-07|typora

    标题：

    ctrl+1(一级标题)[有分割线]

    ctrl+2(二级标题)[有分割线]

    ctrl+数字键(1~6)[标题]

    编辑快捷键：

    ctrl+D(选中单词)

    ctrl+L(选中当前行)

    shift+home(当前行最前面)

    shift+end(当前行最后面)

    ctrl+E(选中当前格式文本)

    ctrl+shift+D(删除当前词)

    ctrl+F或者直接F3(查找) F3(查找里的下一个) shift+F3(查找里的上一个)

    ctrl+H(替换)

    段落快捷键:

    ctrl+shift+[(有序列表)

    ctrl+shift+](无序列表)

    ctrl+shift+K(代码块)

    ctrl+shift+M（公式块）

    格式快捷键:

    ctrl+shift+`(代码)

    this is a test

    插入代码块 ```+语言类型 (java、c++、c、python)

    ctrl+shift+q(引用)

    这里是引用样例

    ctrl+U 测试（下划线）

    alt+shift+S 删除线测试(删除线)

    ctrl+B 测试(字体加粗)

    ctrl+I 测试(字体倾斜)

    ctrl+K(链接) www.baidu.com

    ctrl+shift+I(插入图片)

    ctrl+shift+L(显栏示侧边)

    ctrl+shift+1(大纲视图)

    ctrl+shift+2(文件列表视图)

    ctrl+shift+3(文件树视图)

    ctrl+shift+F(搜索文件)

    ctrl+T(生成表格)

    表格测试

    shift+tab(减少缩进)

    tab(缩进)

    —或者*** + enter(分割线)

    -+space(表示出小圆点)

    无法使用：

    x^2^

    h2o

    X2 H2O

    x^2^ H2O(角标)

    ==高亮== (高亮)

    ctrl+home(到文章开头)

    ctrl+end(到文章结尾)

    (开启拓展后使用)

    复制到dev c++后自动排版(ctrl+shift+a)
